# KakaoSDKUser

[KakaoSDKUser](https://github.com/kakao/kakao-ios-sdk) Swift Package 지원 버전입니다.

## Requirements

- Xcode 13.2.1+
- Swift 5.5+

## Installation

### Swift Package Manager

```swift
.package(url: "https://github.com/jaemyeong/KakaoSDKUser-SPM.git", .upToNextMajor(from: "2.8.4"))
```

## Usage

```swift
import KakaoSDKUser
```
